Risks and Timeline

Risk Impact
Sensors shipping late Medium Impact, we already have one sensor working, we just need to extend what we have.
Sensors too sensitive / not working as expected Potentially Large Impact, time will need to be invested in calibrating them properly.
Robustness of sensor connectivity. Low Impact, expected use case is sensors on ethernet or private network, should be client's responsibility to some extent.
User system not (fully) implemented Low to Medium Impact, single user system is viable to demo functionality.
Usability testing of website not completed Low Impact (testing of code occurs during development).
Downloading large amounts of data from server. Medium to High Impact. We have yet to be able to download and handle large requests to the server since it refreshes so often. We may find that there are issues with accessing and handling large volumes of data in real time.
